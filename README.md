# LANG Project

**[lang-language-project.github.io](https://lang-language-project.github.io/)**

A suite of language tools developed as part of the LANG Project.

---

### [Pragma](https://github.com/albazzaztariq/Pragma)

Declarative pipeline language for data movement and automation. A pipeline is a flat sequence of named actions — fetch, convert, save, notify, and others. No control flow, no variables, no business logic. Describe what moves where and in what format; the runtime handles the rest. Sits above UniLogic and can generate UniLogic code.

### [UniLogic](https://github.com/albazzaztariq/UniLogic)

Unified systems, application, and web language. One syntax across all three — memory management, pointers, and bitwise operations in the same file as high-level logic, with no mode switch. Transpiles to C, Python, and JavaScript. Runtime behaviour (memory model, allocator, overflow, bounds checking, float semantics) is declared per file at compile time, not fixed by the language. Includes a formal memory safety proof system with verifiable certificates.
